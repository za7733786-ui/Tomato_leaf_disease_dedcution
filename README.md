# 🍅 Tomato Leaf Disease Detection using Deep Learning

A deep learning-based project for detecting tomato leaf diseases using image classification techniques.  
This project uses Convolutional Neural Networks (CNN) / Transfer Learning models trained on tomato leaf datasets and is deployed on Google Colab.

---

## 📌 Project Overview

Tomato plants are vulnerable to various diseases that reduce crop quality and yield.  
This project aims to automatically identify tomato leaf diseases from images using Artificial Intelligence and Deep Learning.

The model can classify different tomato leaf conditions such as:

- Tomato Early Blight
- Tomato Late Blight
- Tomato Leaf Mold
- Tomato Mosaic Virus
- Tomato Healthy Leaves
- and more...

---
